# Brain Quest - لعبة الألغاز المتدرجة

## نبذة عن المشروع
تطبيق ويب تفاعلي يقدم تجربة لعبة ألغاز ممتعة وتدريجية الصعوبة. تم تطويره باستخدام أحدث تقنيات الويب لضمان أداء عالٍ وتجربة مستخدم مميزة.

## الميزات الرئيسية
*   **تصميم عصري وجذاب**: واجهة مستخدم مبنية باستخدام Tailwind CSS مع تصميم متجاوب (Responsive) يعمل على جميع الأجهزة.
*   **10 مراحل متنوعة**: تتدرج في الصعوبة وتشمل ألغاز منطقية، لغوية، وثقافية.
*   **نظام تقييم**: احتساب النقاط وتخزين التقدم تلقائيًا.
*   **مؤثرات بصرية وصوتية**: تفاعل حيوي مع إجابات اللاعب.

## التقنيات المستخدمة
*   **React + Vite**: لبناء واجهة سريعة وخفيفة.
*   **Tailwind CSS**: لتنسيق وتصميم الواجهة.
*   **Framer Motion**: لإضافة حركات وانتقالات سلسة.
*   **Canvas Confetti**: لاحتفالات الفوز.

## طريقة التشغيل

1.  تثبيت المكتبات:
    ```bash
    npm install
    ```

2.  تشغيل المشروع:
    ```bash
    npm run dev
    ```
